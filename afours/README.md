# Excel 회계 프로그램 (매입/매출 + VAT + 월별 누적)

## 기능
- 엑셀 파일 업로드 (`.xlsx`, `.xls`)
- 매입/매출 자동 인식 및 부가세 계산
- 월별 집계 + 누적 합계 + 월 손익(공급가 기준)
- 거래처(적요)별 매입/매출 집계
- 기간 필터(시작일~종료일)로 조회/집계/내보내기 범위 제어
- 고정비 자동 추출(매입 반복/금액 안정성 기반 후보 추천)
- 고정비 후보 확정/제외/해제 저장(반자동 운영)
- 결과 엑셀 내보내기(월별요약/거래처별집계/거래원장/고정비후보/고정비판정)
- SQLite 기반 누적 저장 (`accounting.db`)
- 중복 행 자동 스킵(동일 데이터 재업로드 방지)

## 실행
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## 기본 엑셀 컬럼
필수:
- `날짜` (또는 `일자`, `거래일`)
- `구분` (`매입` / `매출`)
- `공급가액` (또는 `금액`)

선택:
- `부가세` (없으면 공급가액의 10% 자동 계산)
- `적요` / `거래처` (거래처별 집계 기준으로 사용)

## 참고
- 업로드할수록 데이터가 계속 누적됩니다.
- 같은 행을 다시 업로드하면 해시 기반으로 중복 저장이 스킵됩니다.
- 고정비 후보 기준: 활성월 3개월 이상, 반복률 60% 이상, 변동계수(CV) 0.2 이하
