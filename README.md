# ERP 회계 시스템 (Streamlit)

## 핵심 기능
- 엑셀 업로드 후 전표/분개 자동 생성
- 계정과목 마스터 관리(추가 가능)
- 월별 매입/매출/VAT/손익 집계
- 전표 원장, 분개장, 합계잔액시산표, 손익계산
- 거래처별 집계, 고정비 자동 후보 추천 + 확정/제외/해제
- 월마감/마감해제(마감월 데이터 업로드 차단)
- 결과 엑셀 내보내기

## 엑셀 컬럼
필수:
- `날짜` (`일자`, `거래일` 가능)
- `구분` (`매입` / `매출`)
- `공급가액` (`금액` 가능)

선택:
- `부가세` (없으면 공급가 10% 자동 계산)
- `거래처` (없으면 적요 사용)
- `적요`

## 자동 분개 규칙
- 매출:
  - 차변 `외상매출금(1200)` = 공급가 + 부가세
  - 대변 `매출(4000)` = 공급가
  - 대변 `부가세예수금(2100)` = 부가세
- 매입:
  - 차변 `매입(5000)` = 공급가
  - 차변 `부가세대급금(1300)` = 부가세
  - 대변 `미지급금(2000)` = 공급가 + 부가세
